# MFC-ImageProcessing
该资源为MFC制作的图像处理工具，采用C++编写，具体功能包括：
- MFC详解显示BMP格式图片
- MFC单文档分割窗口显示图片
- MFC实现图像灰度、采样和量化功能
- MFC对话框绘制灰度直方图
- MFC图像点运算之灰度线性变化、灰度非线性变化、阈值化和均衡化处理
- MFC空间几何变换之图像平移、镜像、旋转、缩放处理
- MFC图像增强之图像普通平滑、高斯平滑、Laplacian、Sobel、Prewitt锐化处理
